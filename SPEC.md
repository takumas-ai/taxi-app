| 条件 | 無料期間 |
|---|---|

## 招待・紹介システム

### 招待コード
- 形式: `TK-XXXXXX`（6文字英数字、0/O/1/I/L除外）
- アカウント作成時に自動生成 → `users.referral_code` に保存
- 共有方法: LINEリンク（`?ref=コード`）＋ 手入力
- 登録画面: コード入力欄あり（リンク経由で自動入力）

### 招待カウントのルール
- 登録完了した時点でカウント（有料転換は条件にしない）
- `referral_events` テーブルで管理（不変ログ）
- `referred_id` に UNIQUE 制約 → 同一人物の二重カウント防止
- ユーザー削除後もレコードが残るのでカウント維持（`on delete set null`）

### 招待した側の特典（マイルストーン）

| 累計招待人数 | 特典 |
|---|---|
| 1人目 | +100 XP ＋ 14日延長クーポン |
| 3人目 | +100 XP ＋ 30日延長クーポン |
| 6人目 | +100 XP ＋ 30日延長クーポン |
| 9人目 | +100 XP ＋ 30日延長クーポン |
| 12人目 | +100 XP ＋ 30日延長クーポン |

※ XP +100 は 1人招待ごとに毎回付与（マイルストーン関係なく）

- 通知方法: アプリ内通知
- クーポン有効期限: **無期限**

### 招待された側の特典
- 登録時に +30日延長クーポン（`type: 'invited'`）が自動発行

## DBテーブル（主要）

### users
| カラム | 内容 |
|---|---|
| id | UUID |
| name | 表示名 |
| email | メール |
| work_type | 勤務形態（隔日/日勤/夜勤） |
| company_name | 会社名 |
| monthly_target | 月間目標売上 |
| closing_day | 締日 |
| area | エリア |
| xp | 累計XP |
| referral_code | 自分の招待コード（TK-XXXXXX） |
| referred_by | 使った招待コード |
| free_period_ends_at | 無料期間終了日時 |
| avatar_url | カスタム画像URL |
| avatar_preset | プリセットアバター番号 |
| streak_days | 連続ログイン日数 |
| last_active_date | 最終アクティブ日 |

### referral_events（招待ログ・不変）
| カラム | 内容 |
|---|---|
| referrer_id | 招待した人のID |
| referred_id UNIQUE | 招待された人のID |
| referral_code | 使用されたコード |
| referrer_name | 招待した人の名前（非正規化） |
| referred_name | 招待された人の名前（非正規化） |

### coupons（クーポン台帳）
| カラム | 内容 |
|---|---|
| user_id | 所有者 |
| code | クーポンコード（EXT-XXXXXX） |
| type | `invited` or `milestone` |
| benefit_days | 特典日数（14 or 30） |
| milestone_at | 何人目で発行か（二重発行防止） |
| issued_at | 発行日時 |
| used_at | 使用日時（NULLなら未使用） |
| expires_at | 有効期限（NULLなら無期限） |

## プロフィール

- 表示名変更 → Supabaseに即時保存（Googleログイン時に上書きされない）
- アバター: プリセット12種 or カスタム写真アップロード（Supabase Storage `avatars`バケット）
- 締日設定: 毎月の締め日（デフォルト15日）
